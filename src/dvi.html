SmartCalc

TameCalc by tamelabe

About

An extended version of a regular calculator, which can be found in the standard applications of each operating system. In addition to basic arithmetic operations, like plus-minus and multiply-divide, the calculator has the ability to calculate arithmetic expressions based on priorities, as well as some mathematical functions (sine, cosine, logarithm, etc.). In addition to calculating expressions, the calculator also supports the use of the variable x and plotting the corresponding function.
